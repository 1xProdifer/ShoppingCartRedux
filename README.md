A React Native mobile application that demonstrates **Redux Toolkit** state management for an e-commerce shopping cart.

## Project Overview

This lab project implements a simple shopping cart with the following features:

- Product list fetched via simulated API call
- Add / remove items from the cart
- Update item quantities
- Calculate total price and item count
- Clear cart and checkout confirmation
- Navigation between product list and cart screens using React Navigation

## Tech Stack

- **React Native CLI**
- **Redux Toolkit** for global state
- **React Redux** hooks (`useSelector`, `useDispatch`)
- **React Navigation Stack**
- **Axios** (for API simulation)

## **Installation**
1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/projectName.git
cd ProjectName
```

2. Install dependencies:

```bash
npm install
```

3. Run the App:

- Android:

```bash
npx react-native run-android
```

- iOS (requires Mac):

```bash
npx react-native run-ios
```
# Note: iOS testing requires Xcode and a Mac.
